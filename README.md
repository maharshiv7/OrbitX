# ORBITX // TACTICAL SPACE RADAR

> *There is a staggering amount of data out there in the dark. We just lack the architecture to see it.*

**OrbitX** is a custom-built, full-stack astronomical intelligence module running entirely on localhost. Engineered to pull the cosmos down to a terminal, it provides an immersive, interactive experience to map, query, and predict celestial data.

---

### ⚙️ THE ARCHITECTURE (TECH STACK)
* **Backend Core:** Python & Flask
* **Frontend Interface:** HTML5, CSS3 (Glassmorphism & Cinematic UI), Vanilla JavaScript
* **Database Engine:** MySQL / PHPMyAdmin (for user handling and querying)
* **Live Data APIs:** Integration for real-time ISS telemetry

---

### 🌌 INTELLIGENCE MODULES (FEATURES)
* **Interactive Solar System:** 3D rendering of planetary bodies with real-time orbital mechanics.
* **Deep Space Archive:** High-fidelity mapping of stellar nurseries, nebulae, and supernovas.
* **Exoplanet Database:** A live-querying engine capable of filtering thousands of alien worlds by radius and distance.
* **Live ISS Tracker & Time Machine:** Real-time positioning of the International Space Station, coupled with a trajectory predictor mapping past and future coordinates.
* **Level-5 Classified:** A restricted-access database decrypting cosmic anomalies and unsolved astronomical mysteries (e.g., The Wow! Signal).
